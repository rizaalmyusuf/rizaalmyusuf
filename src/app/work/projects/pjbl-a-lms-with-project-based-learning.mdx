## Overview

Learning Management System for school or campus with project based learning.

## Key Features

- **Project Management**: Creating, reading, updating, and deleting project record.
- **Student Group Management**: Creating, reading, updating, and deleting student record.
- **Student Task Checklist**.

## Technologies Used

- **CodeIgniter 3**: For fast building server-side system.
- **MySQL**: For managing database application.
- **Bootstrap**: For styling interface application.
