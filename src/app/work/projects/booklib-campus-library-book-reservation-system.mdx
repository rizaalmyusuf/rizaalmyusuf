## Overview

Digital campus library book reservation management system.

## Key Features

- **Book Reservation**.

## Technologies Used

- **CodeIgniter 3**: For fast building server-side system.
- **MySQL**: For managing database applications.
- **Bootstrap**: For styling interface applications.
