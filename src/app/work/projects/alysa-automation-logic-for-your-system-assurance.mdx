## Overview

ALYSA streamlines ISO compliance journey with intelligent document management, GRC automation, AI-powered insights.

## Key Features

- **Document Repository**: Centralized, version-controlled repository with attribute-based access, audit trails, and automated approval workflows.
- **Audit & Risk Management**: Streamlined audit workflows, automated evidence collection, and real-time risk scoring to make audit faster, consistent, and actionable.
- **Collaboration & Reporting**: Task assignment, action tracking, notifications and exportable, auditor-ready reports.
- **Compliance Automation**: Scheduled controls, automated evidence collection, gap detection and remediation flows to reduce manual work and accelerate compliance cycles.
- **Insights & Analytics**: AI-powered insights, KPI dashboards and trend detection for executive reporting and continuous improvement.

## Technologies Used

- **Next.js**: For front-end application system.
- **Golang**: For back-end application system.
- **PostgreSQL**: For data storage application system.
- **MinIO**: For object storage application system.
- **Redis**: For session application system.
- **Docker**: For system infrastucture and microservices.
- **Jenkins and GitHub Actions**: For CI/CD pipelines.
- **SonarQube**: For code analysis and quality gate.
- **OpenTelemetry, Prometheus, Loki, and Grafana**: For monitoring application system.
